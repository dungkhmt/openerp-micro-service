# Hệ thống Quản lý Giao vận 3 Chặng

## Mô hình 3 chặng

Hệ thống hỗ trợ các doanh nghiệp tổ chức hiệu quả hoạt động vận chuyển hàng hóa theo 3 chặng:

- **First Mile (Thu gom)**: Quá trình thu gom hàng hóa từ người gửi
- **Middle Mile (Trung chuyển)**: Vận chuyển hàng hóa giữa các trung tâm phân phối
- **Last Mile (Giao hàng)**: Giao hàng cuối cùng đến người nhận

## Tính năng chính

### Quản lý đơn hàng
- Tạo và theo dõi đơn hàng
- Quản lý trạng thái đơn hàng theo từng chặng
- Phân công nhân viên tự động

### Quản lý vận hành
- Thu gom hàng hóa
- Trung chuyển giữa các hub
- Giao hàng cuối cùng
- Quản lý các tuyến đường

### Quản lý tài nguyên
- Quản lý nhân sự
- Quản lý các trung tâm phân phối
- Tối ưu hóa phân công với 2 thuật toán linh hoạt

## Công nghệ sử dụng

- **Frontend**: ReactJS
- **Backend**: Spring Boot
- **Database**: 26 bảng cơ sở dữ liệu
- **Platform**: Web-based application

## Thống kê dự án

- **48 use cases** bao quát toàn bộ quy trình vận hành
- **26 bảng** cơ sở dữ liệu
- **2 thuật toán** phân công đơn hàng tối ưu
- Hỗ trợ đầy đủ các nghiệp vụ cốt lõi của ngành logistics

## Lợi ích

- Tối ưu hóa hiệu suất vận hành
- Nâng cao năng suất lao động
- Cải thiện mức độ hài lòng của nhân viên và khách hàng
- Linh hoạt điều chỉnh theo nhu cầu doanh nghiệp
